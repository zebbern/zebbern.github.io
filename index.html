What is Claude Code?

Claude Code is Anthropic's agentic AI coding assistant that lives in your terminal, understands your entire codebase, and accelerates development through natural language interactions.

Terminal Native

Works directly in your terminal with full codebase understanding. No context switching between tools.

Claude 4 Powered

Powered by Claude Opus 4 and Sonnet 4 - the world's best coding models for complex, long-running tasks.

MCP Integration

Connect to remote MCP servers for enhanced capabilities. Access external tools and data sources seamlessly.
